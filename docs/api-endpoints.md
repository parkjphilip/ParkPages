# API Endpoints

## HTML API

### Root

- `GET /` - loads React web app

## JSON API

### Users

- `POST /api/users`

### Session

- `POST /api/session`
- `DELETE /api/session`

### Parks

- `GET /api/parks`
- `POST /api/parks`
- `GET /api/parks/:id`
- `PATCH /api/parks/:id`
- `DELETE /api/parks/:id`

## Reviews
- `GET /api/park/id/reviews`
- `POST /api/park/id/reviews`
- `PATCH /api/park/id/reviews/:id`
- `DELETE /api/park/id/reviews/:id`
